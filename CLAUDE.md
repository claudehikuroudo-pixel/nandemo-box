# なんでも箱（気になる箱）— 開発セッション用ガイド

このフォルダは「なんでも箱（気になる箱）」アプリの開発プロジェクト。
SNS戦略セッションとは分離した、**開発専用セッション**でここを開いて作業する。

## まず読むもの
1. **`要件定義書_2アプリ.md`** … これが開発の起点。何を・どの順で・どう作るかが全部書いてある。
2. `nandemo_box_live.jsx` … 実際にClaude APIで仕分けが動くモック（心臓部の参考実装）
3. `nandemo_box_scheduler.html` … 確定UI方向（上カレンダー＋下ウォールの2層）

## このアプリの要点
- 気になるもの（スクショ/URL/テキスト）を**放り込む**→ AIが**5分類（予定/タスク/モノ/場所/情報）に自動仕分け**→ 上カレンダー＋下ウォールで見返す。
- 思想：**Notionの逆**（設計させない・入口1アクション）。見返さなくていい（必要な時に引く）。カレンダー常時表示で毎日開く理由を作る。
- **最優先実装＝写真・スクショの放り込み**（Claude Vision）。ここが動かないと製品の意味がない。

## 技術スタック
React(PWA) / Claude API(Haiku・Vision) / 保存はまずローカルストレージ→後でSupabase / Cloudflare Pages / Stripe課金。
⚠️ PWAはiOSで他アプリからのシェア受け取り不可。写真選択・URL・テキストはPWAで動く。シェアはApp Store版(v3)。

## MVPで作るもの（今週末スコープ）
1. 週カレンダー常時表示＋予定の手動追加
2. 放り込む（写真/URL/テキスト）→ AI仕分け → ウォール＋タブ表示
3. ローカル保存
※ Googleカレンダー同期・浮上通知・レコメンド・シェア受け取りは v2以降。
