# なんでも箱（気になる箱）

気になるもの（写真・スクショ・URL・テキスト）を**放り込むだけ**で、AIが自動で5分類（予定・タスク・モノ・場所・情報）し、上カレンダー＋下ウォールで見返せる、ライトユーザー向けの「情報おもちゃ箱 兼 スケジューラー」。

## 使い方
1. `index.html` をブラウザで開く
2. 右上の ⚙ から Anthropic（Claude）の API キーを設定
3. 「放り込む」からテキスト・写真を入れると AI が自動で仕分け
4. 予定・タスクは本文の日付を AI が読み取り、カレンダーに自動反映

## 技術
- 単一HTMLファイル（React は CDN 経由・ビルド不要）
- Claude API（Haiku / Vision）で分類
- データは端末内（localStorage）に保存。サーバーに送られるのは分類時の本文・画像のみ

## スマホで使う
GitHub Pages 等に置いて URL を開き、「ホーム画面に追加」で PWA として利用可能。
