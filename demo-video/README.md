# デモ動画 自動編集パイプライン（人力編集ゼロ）

**あなた＝素材を撮るだけ。編集は全部自動。**

## 使い方（毎回これだけ）
1. iPhoneで**画面収録**（撮って出しでOK。多少長くても可）
2. その動画をこのPCの `demo-video/incoming/` に入れる（AirDrop→保存、LINE経由等）
3. Claudeに「**動画作って**」と言う → テロップ・BGM・縦整形・書き出しまで自動 → `output/` に完成品

## 初回だけ（1回やれば以後ずっと自動）
- `bgm/` フォルダに**好きなBGMを1曲**入れる（mp3/m4a/wav）。
  入手先例：Canvaのオーディオ素材・YouTubeオーディオライブラリ（無料・商用可のもの）。
  ※入れておけば毎回自動でBGMが付く（音量0.45・終わり2秒フェードアウトも自動）。

## 仕組み
- `make_demo.ps1` … ffmpeg自動編集（縦1080x1920化・テロップ焼き込み・BGM合成・X向けmp4）
- `telops.json` … テロップと表示タイミング（sとeは秒 or 0〜1の割合）。動画の内容が変わったらここだけ直す
- 実行例：`powershell -ExecutionPolicy Bypass -File make_demo.ps1 -In incoming\xxx.mp4`

## 現在のテロップ（予定自動記入デモ用）
1. 放り込むだけ（冒頭〜22%）
2. AIが いつ・どこ を読む（22〜55%）
3. 予定になって戻ってくる（55〜85%）
4. いつ蔵（85%〜最後）
